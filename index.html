  • Experience

    • SoundUp - Real Project (1 month ago)

      SoundUp is a sound enhancement software developed for a sound and amplification organization in the city of Barranquilla, using Java and Maven. The project was carried out in a team of 5 people, where my role included tasks related to the calculation and visualization and separation of frequency ranges, as well as noise reduction. Advanced techniques such as dynamic and cumulative superposition by frequency and signal enhancement using the RMS index were applied. These techniques were essential to optimize the sound quality, improving both the clarity and the power of the acoustic signals.

    • Tweet Analysis - Semester 7

      This project focused on analyzing a massive database with over 1,000,000,000 Twitter tweets, using Python. Parallelization techniques were employed to optimize large-scale data processing. The goal was to examine the relationships between users across their tweets and retweets, and visualize these relationships in a graph. The nodes in the graph were sized and colored based on the strength of the relationship, allowing for a clear representation of influence and information propagation on the platform. This project facilitated a deep understanding of social interactions on Twitter and its influence dynamics.

    • PinterFull - Semester 5

      PinterFull is a Java project that uses threads and sockets to create a collaborative online gaming experience. Players connect to a drawing board where one takes on the role of an artist, drawing an object or word while the others try to guess it via a chat. The implementation of threads ensures that actions are handled concurrently, and sockets facilitate real-time communication between participants. The project stands out for its ability to offer an interactive and dynamic experience, promoting both competition and collaboration.

    • Spaces Invaders Evolution - Semester 3

      Spaces Invaders Evolution is an innovative reinterpretation of the classic Space Invaders game, created during the pandemic. Instead of traditional spaceships and aliens, the game's ship is a syringe and the enemies are viruses, reflecting the global health context. The game features an integrated store where users can register, earn virtual currency, and purchase upgrades and customizations for their ship. Developed in Java with data management in SQL, the project not only offered an entertaining gameplay experience, but also introduced customization and trading elements, enriching the user experience and adapting to the context of its creation.
